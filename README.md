# Underhell
A survival horror game set in a mysterious underground facility. This game is completely inspired by [Undertale](https://undertale.com/).

## Quick Start
1. Clone repo
2. Make sure you have JDK 17+ and JavaFX SDK installed (prefer to install [bellsoft full JDK](https://bell-sw.com/pages/downloads/#jdk-21-lts))
3. Run the project from MainApp

## Controls
- Controls: Move
